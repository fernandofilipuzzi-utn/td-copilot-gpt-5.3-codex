# Formato de Entrega Requerido

Estructura de carpeta comprimida con entregables:
1. Informe final en PDF (max 50 paginas, Arial 11, interlineado 1.15)
2. Scripts de simulacion (Octave/MATLAB o Python segun variante)
3. Documentacion del agente de IA
4. Codigo embebido (C/C++/Arduino/MicroPython)
5. Graficos de simulacion (PNG/PDF)
6. Dataset (opcional)
7. Video de presentacion y demostracion (obligatorio)
8. Presentacion (PPTX/PDF)

## Secciones obligatorias del informe
- Caratula
- Resumen ejecutivo
- Indice
- Marco teorico
- Analisis del problema
- Diseno de filtros
- Simulacion
- Agente IA/sistema experto
- Implementacion embebida
- Conclusiones
- Bibliografia (IEEE, minimo 5 fuentes)
- Anexos (opcionales)
